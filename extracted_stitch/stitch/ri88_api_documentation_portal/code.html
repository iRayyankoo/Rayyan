POST /v1/users

Create a new user account

This endpoint allows you to programmatically create a new user on the RI88 platform. All users created via API will require email verification by default.

list_alt Request Parameters

email STRING REQUIRED

The primary email address for the account.

username STRING

Unique identifier between 4-20 characters.

code Example Request

request.json 
{
  "email": "dev@ri88.io",
  "username": "ri88_master",
  "metadata": {
    "role": "administrator",
    "tier": "platinum"
  }
}
check 201 Created

The request was successful and the user entity has been initialized. You can now use the returned id to manage the user profile.
